Keystroke Library tutorial

First of all, you'll want to make the basic folder structure.


-pages

--[library name] (I'll call this the main folder from now on)

---images (This is where you'll put all your images)

--[library name].txt (I'll call this the main file from now on)


Open up the main file. On the first line, you'll put the name of your library.

On the second line, you'll put the name of the main folder.

This file will let you load the library with Keystroke.


Next, open Keystroke and navigate to Tools->Page Designer. This is where you can make the pages.

(Make sure to put all your images in the images folder!)


Alternatively, you can make a page using a text editor. Use this template (remove the brackets):

[image file without path] (if there is no image, type "none") [title] [info]
