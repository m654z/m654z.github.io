PyBot

PyBot is a simple chatbot written in Python that can be easily expanded and built on.

Like AIs, PyBot learns from the user, so it can demonstrate machine learning.

Creating a simple bot

Here's an example of a chatbot made using PyBot.

import pyBot
myBot = pyBot.Bot()
while 1:
	myBot.evaluate(input("Input > "))

Pretty easy, eh?
