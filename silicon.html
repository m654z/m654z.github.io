Silicon

Silicon is a programming language of mine I made for code golf challenges.

Silicon is an esoteric language, so it might be a little hard to learn. It has many, many one-character commands to keep the code short.

Silicon was inspired by many other code-golfing languages, including Pyth and Seriously.


Getting started

To get started with Silicon, head over to its GitHub page and clone the repository.

Silicon has a built-in interpreter, but files with Silicon code can be run from the command line.

The easiest way to get started with Silicon is downloading the Silicon IDE, which includes the actual programming language.

It makes writing and running program much easier. You can download it here. It isn't a virus, for your information. Go do a VirusTotal scan, whatever.
