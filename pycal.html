PyCal

PyCal is an esoteric programming language designed for code-golf challenges, especially math challenges.

It includes many useful math-related commands, so winning (or coming close to winning) should be quite easy.

PyCal is very easy to learn, so why not try it out now?

Examples

Output the first n prime numbers 
IP!

Calculate (8*62)/2 
{(8*62)/2}!

Output the square root of 2 
(2)S!

Get the mode of a list 
(2,7,3,22,8,3,7,6,3)sO.!
