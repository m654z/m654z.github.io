Python tutorial

Part 1 - Getting started

You’re ready to learn Python? Great!

But wait, don’t get too excited yet. I’m assuming you don’t have Python on your computer yet.

Note: Some Linux distributions come with Python pre-installed.

Installing Python is quite simple. Head over to Python’s homepage at python.org. Go over to the Downloads section, and download Python 3.4.4 for your operating system.

Once you have Python installed, go to the Start Menu (or the equivalent for your operating system) and look for a program named something like “IDLE (Python 3.4 GUI)”. Open it up.

You’ll see something like this:

This is the “Python Shell”, sometimes known as “Interactive Python”.

The Python Shell is great for writing simple programs, as the code is executed (run) right away, but for longer ones you’ll need to create a separate file.

Info: You can make new files by going to File -> New.

For now, though, we’ll be using the Python Shell.

When you learn a new programming language, the first program you write is a program that says “Hello, world!” Let’s do it in Python!

So, you’ve got the Python Shell open. Type in the following, and press Enter:

print("Hello, world!")

Did you see what happened? Python spoke to you! To make python “print” things onto the screen, use the print() command.

Feel free to experiment with the print() command some more!

Well, displaying text is good and all… but what if you want to do some math?

Don’t worry, Python is excellent at math. Let’s try a few simple calculations in the Python Shell.

2+2 should output 4.
8*8 should output 64.
14-6 should output 8.
32/8 should output 4.

Note: expressions like 2+2 work fine in Interactive Mode, but while working in a file, you’ll have to use print(2+2).

As you might have noticed, some of the operators (+, -, etc.) are different from the ones you usually use. For example, /. Unfortunately, there’s nothing you can do about it, so you’ll just have to get in the habit of using the new operators.

Well, that concludes Part 1, but before you move on to the next part, you’ll want to do these simple exercises.

  • Experiment with the print() command. Try doing print("text" * 10) and see what happens!
  • Try calculating something useful with Python!
  • Write a simple program that combines the print() command and math.
