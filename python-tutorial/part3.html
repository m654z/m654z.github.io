Part 3

So far, you’ve been writing output-only programs. That means that the programs only display text. They aren’t very interactive.

Well, let’s change that. Let’s get to know a new command, input(). input() asks for, well, input. Let’s make a program that asks for the user’s name is, and says “Hello, (name)!”.

print("What is your name?")
name = input()
print("Hello, " + name)

Yes, you can assign input to variables. In this case, the user’s input is being assigned to the name variable.

Info: If you don’t want to use print() along with input(), you can also do input("What is your name? "). This gets input on the same line as the text.

The next line of code displays “Hello, ” along with the name variable.

To add two or more strings together, you can do print(text1 + text2). It’s as simple as that. Of course, if you try adding two integers together, you’ll get the result, but we’re talking about strings.

Well, now that you know about displaying text, math, variables and input, let’s combine those and create a simple calculator (more specifically, an adder).

Before you freak out, let me tell you that it’s actually quite simple. Type the following code into a file:

num1 = input("Type the first number: ")
num2 = input("Type the second number: ")
print("The result is " + str(num1 + num2) + ".")

Hey, why is there a str() command there? Well, you can’t add integers to strings, so you have to convert the integer to a string first. str() does just that.
