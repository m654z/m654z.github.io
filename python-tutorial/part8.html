Part 8

In this part, we’ll be learning about functions. Functions are snippets of code that can be called (run) from anywhere in the code. It keeps the code clean and hopefully more readable.

Let’s define a simple function using the def command.

def simpleFunction():
    print("Hello")

This function can be called from anywhere in the code like this:

simpleFunction()

The above code will output “Hello”.

The parentheses are for arguments. Arguments are simply variables that the function will do something with. Arguments go in between the paretheses, as so:

def greet(name):
    print("Hello, " + name)

Can you guess what it does? Try calling this function.

greet("John")

You can have as many arguments as you like:

def greet(name1, name2, name3):
    print("Hello, " + name1 + ", " + name2 + " and " + name3)
    
greet("John", "Mary", "Bob")
