Part 4

So far, your programs haven’t done any decision-making. Let’s say you wanted to make a program that asks for two numbers, and checks if they are equal.
If they are equal, the program displays “Equal!” If they aren’t, it displays “Not equal”.

For this, you’ll need to use conditional statements, also known as “if statements”.

In Python, the basic syntax for if statements is:

if (condition):
    do this

If the condition is True, the code will be executed. If it isn’t, the code will be skipped.
What if you want different code to run if the condition is False? Use an if-else statement.

if (condition):
    do this
else:
    do this

Let’s write the number-checking program using an if-else statement.

number1 = input("Input a number: ")
number2 = input("Input another number: ")

if number1 == number2:
    print("Equal!")
else:
    print("Not equal.")

The code is quite self-explanatory.
Hmm, what’s the ==? Why can’t you use = instead?

Well, that’s because = is used for assigning variables to data. == is a comparison operator. Comparison operators are used to compare different data.

  • == checks if two values are equal. If they are, the condition is True.
  • != checks if two values are not equal. If they aren’t, the condition is True.
  • > checks if the left value is greater than the right value. If they are, the condition is True.
  • < checks if the left value is less than the right value. If they are, the condition if True.
  • >= checks if the left value is greater or equal to the right value.
  • <= checks if the left value is less or equal to the right value.

You can compare any type of data using comparison operators:

if "abc" == "cba":
    print("Condition is True!")
else:
    print("Condition is False!")

Or:

if 10.6 > 9.3:
    print("Condition is True!")
else:
    print("Condition is False!")

Info: Numbers with decimal points such as 10.6, 3.7 and 6.542 are called floating-point numbers, or simply floats.

Exercises

  • Write a program that asks for the user’s favorite number. If it’s over 50, print “Your favorite number is over 50.” If it isn’t, print “Your favorite number is under 50.”
  • Make a simple guessing game. Choose a number. Then, get input, and if the input is equal to the number, display “You won!”

Note: Remember that you can’t compare integers to strings! You can use int() to convert a string to an integer, and str() to convert an integer/float to a string.
