Last part was pretty simple, but now we’re going to learn about something more advanced.

print() is great for printing text, but what if you want to display some text two times in the code?

If you thought “Well, use print("text" * 2), duh” then congratulations! You’re right.

But what if you want to display the same text two times in different places? You’ll probably want to use variables.

Variables are, simply explained, pieces of data assigned to a name. For example, you could have a variable named “book” that is assigned to the text “The Lord of the Rings”.

In Python, you would do this:

book = "The Lord of the Rings"

Now, if you want to display book’s data, you could do print(book). Hey, no quotes! Well, when printing variables, you don’t need quotes.

You can even do this with variables:

a = 10
b = 20
print(a + b)

What do you think this outputs? It outputs 30, because you are adding the variables a and b together, which are assigned to 10 and 20.

You might have noticed that book = "The Lord of the Rings" has quotes, but a = 10 doesn’t.

Well, that’s because The Lord of the Rings is a string.

Strings are, simply, text. Let’s say that a string is something that you can’t add to an integer (which is a number).

So, strings need quotes, but integers don’t.
