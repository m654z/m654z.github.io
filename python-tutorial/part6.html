By now, I hope you know what a variable is, because in this part we’ll be learning about lists.
Lists are, simply explained, variables that can store multiple items.

Let’s say you wanted to store the number from 1 to 5 in a list. In Python, you would do this:

numbers = [1, 2, 3, 4, 5]

As you can see, the list’s items are put in between the square brackets [ ], and they are separated by commas.

Lists can contain any type of data:

data = ["Hello", 5, "z", 8.252]

They can even contain other lists:

lots_of_lists = [[1, 2, 3], ["Hi", "there"]]

But how do you use the lists? Well, to get a specific item from a list, you can use the syntax name[item]. For example:

myList = ["Hello, world!", "abc"]
print(myList[0])

The above code will display the first item in the list, “Hello, world!”

Note: List indices start from 0, not 1.

An alternative would be assigning a list item to a variable, and then printing the variable:

myList = ["Hello, world!", "abc"]
myItem = myList[0]
print(myItem)

What if you want to add an item to a list while the code is running? You should use the append() function. (We’ll talk more about functions later, but for now just treat it like any other command)

myList = []
myList.append("Hello")
print(myList)

The code would display ["Hello"]. As you can see, you can use append() to add an item to a list any time.

Let’s write a simple program that gets two numbers, adds them to a list and displays them.

numbers = []

num1 = input("Choose a number: ")
num2 = input("Choose another number: ")

numbers.append(num1)
numbers.append(num2)

print("You chose " + numbers[0] + " and " + numbers[1])

Try it out!

You can also insert data wherever you want in the list using the insert() function.

myList = myList.insert(6, "Hi")

Would insert “Hi” at the sixth position in the list.

Sometimes, you’ll need to remove items from a list to. There are three functions that can do this, but they all do different things to the list.

  1. del()
  2. remove()
  3. pop()

del()

del() lets you remove an item from wherever in the list using the item’s position in the list. For example, if you wanted to remove the third item in the list, you could write the following code:

list = list.del(2)

Why do you need to do list = list...?
Well, del() is a function which returns a value. What you’re doing is list with the third item removed to the variable name list. Yes, it’s a little confusing, but that’s how it works. (More on functions later)

remove()

remove() removes an item from wherever in the list using the item’s data. Let’s say you have the item “Delete this” in the list. To remove it, you would do:

list = list.remove("Delete this").

pop()

pop() is a little different from the other functions. It takes the list’s last item and returns it. You can assign the item to a variable too.

lastItem = list.pop()
print(lastItem)

Would remove the last item from the list, and then display it.
