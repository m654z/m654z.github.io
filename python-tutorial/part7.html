As promised, in this part we’ll be learning more about lists.

Sometimes, you’ll need to join a list, meaning converting it to a string. That’s what the join() function is for. Try running this code:

testList = ['Thi', 's is ', 'a li', 'st!']
testList = ''.join(testList)
print(testList)

It should output “This is a list!”

Basically, the join() function joins a list, and puts whatever is in the quotes between the list items. It’s a little confusing, but it’s not difficult.

Let’s try something else with the join() function.

testList = ['This is ', ' list!]
testList = 'a'.join(testList)
print(testList)

As you can see, the “a” is put in between the list’s items.

Now, onto the list() function. What it does is convert a string (or object, onto that later) into a list.

print(list("Hello"))

You should see ["H", "e", "l", "l", "o"] appear.

Pretty easy to understand, it basically splits the string at every character.

Now, we’ll learn about the range() function. (If you’re confused about functions, just think of them as normal commands. We’ll learn about them in the next part.)

Remember when we were learning about loops? We used range() back then.

What range() does is create a list of numbers in a certain range. Let’s create a list with the numbers from 1 to 100.

numbers = list(range(1, 100))
print(numbers)

It should output a list with the numbers from 1 to 100.

But why do we need the extra list()? Well, range() returns an object (we’ll learn about them much later) and it needs to be converted to a list.
